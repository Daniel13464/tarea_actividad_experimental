# Sistema de Gestión de Investigación con SQLAlchemy ORM

Sistema para gestionar instituciones, departamentos, investigadores y publicaciones académicas.

## Características
- Gestión de Instituciones y Departamentos
- Registro de Investigadores
- Control de Publicaciones
- Base de datos SQLite
- Interfaz de línea de comandos

## Tecnologías
- Python 3.x
- SQLAlchemy ORM
- SQLite

## Instalación
```bash
pip install sqlalchemy
python main.py
